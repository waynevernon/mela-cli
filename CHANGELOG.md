## [1.0.1] - 2026-03-21

### Changed

- Snazzier terminal output: ANSI color, Unicode rules, ★/◎ indicators, mini bar charts, section headers
- `show` text view: compact metadata block with `·` separators, blank lines between instruction steps
- `list` table: separate F and W indicator columns, dim PK, bold favorite titles, cyan tags
- `tags` table: mini bar chart relative to max count, cyan tag names
- `stats` table: bar charts with percentages for ratio fields
- `doctor` report: grouped sections, `✓`/`✗` status marks, yellow `⚠` warnings
- Help text: descriptions, per-format explanations, and examples for all subcommands; env var epilog annotated
- `search` subcommand description clarified as alias for `list --query`
